# Low-Resource Amazonian NMT

Comparative study of neural machine translation (NMT) models and evaluation metrics for low-resource Amazonian languages using mBART50, mT5 and NLLB.

This repository contains the preprocessing, tokenization, tokenizer augmentation, training and evaluation pipelines developed as part of the thesis:

> *Estudio comparativo de modelos de traducción automática neuronal y métricas de evaluación en lenguas amazónicas de escasos recursos*

---

## Objectives

- Analyze neural machine translation models for Amazonian low-resource languages.
- Compare evaluation metrics for morphologically complex languages.
- Implement syllabification-based tokenizer augmentation strategies.
- Evaluate NMT performance using BLEU, ChrF++ and COMET.
